**Retail Sales Management System**

This project contains a backend (Node.js + Express) and a frontend (React + Vite) for processing and displaying sales data from a large CSV file.

Dataset Requirement

Inside the backend folder, create a new folder named:

backend/data


Download the dataset from the provided link and place it inside the data folder.
'https://drive.google.com/file/d/1tzbyuxBmrBwMSXbL22r33FUMtO0V_lxb/view?usp=sharing'
Rename the dataset file to:

sales.csv


The backend expects the dataset at the following path:

backend/data/sales.csv


Any other filename or location will cause the backend to fail.

Backend Setup (Node.js + Express)
1. Navigate to backend folder
cd backend

2. Install dependencies
npm install

3. Start the backend server
npm run dev


The backend will start at:

http://localhost:4000

Frontend Setup (React + Vite)
1. Navigate to frontend folder
cd frontend

2. Install dependencies
npm install

3. Start the frontend development server
npm run dev


The frontend will start at:

http://localhost:5173
